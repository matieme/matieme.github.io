01

Outsider
UNITY / C# / 3DSMAX
Outsider was developed as the final project for my degree. The final project for this degree consists of creating a game with a group of people, where each member takes care of a different aspect of the game (in this case I was in charge of the game design). After finishing the course of studies, we presented the game idea and prototype in the AGS (Argentina Game Show) which is a big gaming festival in the country. The entire project is on Unity with C#.

Super Dark Colorete
UNITY / C#
Super Dark Colorete is a puzzle game where you have to use the blue fire to activate totems and make your way to get all the gems and move forward to the next level. This project was made for my YouTube channel in which I made one game a month. During this month, I developed the art, the code and the design with Unity and C#.

03

Ghost Roads
UNITY / C#
Ghost roads is a project I did for a final of a subject during my degree in DaVinci. This game consists of creating a compulsion loop through the character's movement, who is a ghost, and avoid colliding against the walls or the obstacles, trying to get as far as possible without losing. The game was developed in two weeks in Unity and C#.

Package Rush 3D
UNITY / C#
This project was developed together with Spiky Games .
